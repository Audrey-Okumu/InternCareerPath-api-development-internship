IP Threat Detection API

Base URL

http://localhost:8000/api/v1/

Authentication

Use Bearer Token:

Authorization: Bearer YOUR_API_KEY

Endpoint

POST /ip/analyze/

Example Request (cURL)

curl -X POST http://localhost:8000/api/v1/ip/analyze/ \
-H "Authorization: Bearer sk_live_xxx" \
-H "Content-Type: application/json" \
-d '{"ip":"8.8.8.8"}'

Example Response

{
  "ip": "8.8.8.8",
  "country": "United States",
  "city": "Mountain View",
  "latitude": 37.4056,
  "longitude": -122.0775,
  "isp": "Google LLC",
  "risk_score": 20
}

Error Codes

  • 400 – Invalid request
  • 401 – Invalid API key
