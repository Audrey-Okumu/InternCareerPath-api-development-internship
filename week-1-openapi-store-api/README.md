This repository contains the OpenAPI (Swagger) specification for a Store API.
The API is designed following REST principles and documents how clients can
manage products in an online store.

## Features
- Retrieve all products
- Retrieve a product by ID
- Create a new product
- Update an existing product
- Delete a product

## API Design Highlights
- RESTful endpoints
- Proper HTTP status codes
- Reusable schemas
- Error handling

## Tools Used
- OpenAPI 3.0
- Swagger Editor
